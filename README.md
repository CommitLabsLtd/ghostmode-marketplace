# ghostmode-marketplace

Claude Code marketplace for the [GhostMode](https://github.com/CommitLabsLtd/GhostMode) plugin.

**Status:** pre-alpha. Not for general use yet.

## Install

In a Claude Code session:

```
/plugin marketplace add github:CommitLabsLtd/ghostmode-marketplace
/plugin install ghostmode@ghostmode-marketplace
```

Then run `/ghostmode:init` in your project to scaffold `.ghostmode/`.
